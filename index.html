    The Tensor C++ library

    Tensor is a C++ library for performing numerical computations and linear algebra with multidimensional arrays, also called "tensors". Some of the features of the library are:

    • Free under a liberal license.
    • Autoconfigurable library, using GNU tools
    • Recognizes and uses different libraries such as Apple's VecLib, the ATLAS optimized libraries for matrix and vector operations, or Intel's MKL library.
    • Arrays with any number of indices.
    • Specialized versions for double precision real and complex arrays.
    • Singular value decompositions, computation of eigenvalues.
    • Efficient tensor contraction routines implemented on top of optimized BLAS kernels.
    • Extremely portable.
    • Automatically tested on a variety of platforms using the GoogleTest framework.
    • In addition to this, as time permits, the library will incorporate extra components for numerical computations with Matrix Product States and related algorithms.
